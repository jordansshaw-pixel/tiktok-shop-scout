TikTok Shop Product Scout — Folder-Based AI Operator

Drop the folder into a Claude project or point Claude Code at it. Paste any TikTok Shop product listing — a URL, screenshot, or whatever details you have — and the operator returns exactly one decision.

  • PROMOTE: the product scored against a six-dimension rubric, passed compliance, and ships with a category-aware ready-to-film script.
  • SKIP: the specific rule that killed it is logged in one line.
  • WATCHLIST: the exact signal that flips the product to PROMOTE is named, with the date to check it.

Swap reference/audience-profile.md and the operator redeploys for a different creator. One folder. One decision. Every time.
