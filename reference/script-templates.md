# Script Templates

Use this file only after a product reaches **PROMOTE** under `rules.md`.

A PROMOTE output includes the content brief from `reference/output-templates.md` plus one ready-to-film script generated from the matching category scaffold below.

## Category ID Table

| Category ID | Use when product is primarily... | Compliance priority |
|---|---|---|
| `BEAUTY_SKINCARE` | beauty, skincare, hair, cosmetic tools, makeup, SPF, patches | avoid guaranteed appearance outcomes |
| `KITCHEN_HOME` | kitchen gadgets, food prep, food storage, cleaning helpers | show observed use, not durability claims |
| `HOME_ORGANIZATION` | bins, racks, closet tools, small-space storage, decluttering products | show before/after honestly |
| `FASHION_ACCESSORIES` | bags, wallets, jewelry, clothing accessories, outfit helpers | avoid luxury/dupe claims unless listing proves it |
| `WELLNESS_ADJACENT` | massage tools, posture tools, sleep masks, ice rollers, red-light cosmetic tools, comfort products | strictest non-medical wording |
| `TECH_GADGETS` | chargers, lights, mini appliances, electronic accessories, desk gadgets | no unverifiable specs |

**Override rule:** if a product spans beauty and wellness-adjacent, use `WELLNESS_ADJACENT`. The stricter compliance rules win.

## Global Script Rules

- Target 120-160 words.
- Hard ceiling: 180 words.
- Fill every placeholder before output. No bracketed or placeholder text may remain.
- First sentence is the hook and must address a specific viewer pain point directly.
- Final line is always: `It's linked in my TikTok Shop. Tap the tag. #ad #TikTokShop #[niche tag 1] #[niche tag 2]`
- The final CTA line counts toward the word ceiling.
- No stage markers in the final script. The output should read as words the creator can say on camera.
- No outcome promises. Describe what the camera shows or what the creator observed, not what the product guarantees.
- If the script mentions price, keep the price accurate to the listing or input.

## Mandatory Compliance Pre-checks

Run these before outputting the script.

### Wellness-adjacent

Rewrite before output if the draft contains any of these banned phrases or claims:

- "fixes"
- "treats"
- "relieves pain"
- "doctor recommended"
- "clinically proven"
- any outcome with a timeline

Allowed framing: comfort, setup, visible use, routine, cosmetic observation, or "felt" language without promising the viewer's result.

### Tech

Rewrite before output if the draft contains:

- mAh claims
- wattage claims
- "military grade"
- waterproof, output, battery-life, or performance claims that are not physically demonstrable on camera

Allowed framing: what the viewer can see in the demo, what ports/buttons/lights are visible, setup steps, fit, size, and practical use.
