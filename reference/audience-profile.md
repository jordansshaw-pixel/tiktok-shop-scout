# Audience Profile (ICP)

This file defines who the operator is scouting for. The "ICP fit" rubric dimension scores against it. **Swap this file to redeploy the operator for a different creator** — nothing in `rules.md` changes.

## Default profile

**Primary:** US women 18–34, scrolling TikTok for entertainment, shopping on TikTok Shop as a side effect of stopping. **Secondary:** women 35–45, same behavior, slightly higher price tolerance.

**Spending behavior:** comfortable at $15–$60 without hesitation. Hesitates at $80+. Will go above $60 only if the "why" is unmistakable in the video.

## Buying triggers (what a 4–5 ICP-fit score must cite)

| Trigger | Why it works |
|---------|-------------|
| "I didn't know this existed" | Novelty — immediate share impulse |
| "I've been doing this wrong" | Ego challenge — must resolve the dissonance |
| "This solved [specific problem]" | Proof beats claims |
| "It's only $X" | Price anchoring removes final friction |
| Visible before/after or instant result | Visual proof > verbal promise |

## High-converting categories (in order)

1. Beauty & skincare (serums, tools, patches, SPF)
2. Kitchen gadgets (time-savers, aesthetic items)
3. Home organization (satisfying, low-cost)
4. Health & wellness — *non-ingestible only, see disqualifiers*
5. Fashion accessories (under-$30 items)
6. Pet products (impulse category)

## What kills conversion for this audience

- Hard-sell language and fake urgency
- Overly polished, ad-like presentation
- Health claims that sound too good
- Products whose benefit isn't visible in the first 5 seconds of a demo
